# alarm-clock-app

### Overview
This web application is a simple alarm clock built using HTML, CSS, and JavaScript. It allows users to set alarms, view a list of all set alarms, and receive notifications when alarms are triggered.

### Usage

<ol>
  <li>
    Set an Alarm:
    <ul style="list-style-type:circle;">
      <li>Enter the desired time, title, and message.</li>
      <li>Click the "Set Alarm" button.</li>
    </ul>
  </li>
  <li>
    View Alarms:
    <ul style="list-style-type:circle;">
      <li>The set alarms will be displayed in a list.</li>
    </ul>
  </li>
  <li>
   Delete Alarms:
    <ul style="list-style-type:circle;">
      <li>You can delete alarms from the list.</li>
    </ul>
  </li>
</ol>

### Features
<ul>
<li>Set alarms with customizable time, title, and message.</li>
<li>Display a list of all set alarms.</li>
<li>Allow user to delete any set alarm</li>
<li>Trigger notifications when alarms go off.</li>
</ul>

### Technologies Used
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
